# JLPT N1 错题复盘页面

直接打开 `index.html` 就能使用。

错题截图已复制到 `assets/screenshots`，网页详情页会自动显示对应原图。

新增错题可以在页面里的“截图导入”区域完成：选择或拖入截图后，点击“保存到错题卡片”。这些新增卡片会先保存在当前浏览器里，并标记为“待分析”。“导出新增数据”可以把本地新增内容复制出来做备份。

以后添加错题时，编辑 `questions.js` 里的 `mistakes` 数组，按已有格式继续新增即可。每一道错题会自动出现在卡片区和复盘总表里。

字段说明：

- `id`：题号或自定义编号
- `type`：文字词汇、文法、读解、听解
- `source`：来源或截图说明
- `prompt`：题目重点
- `userAnswer`：你的答案
- `correctAnswer`：正确答案
- `focus`：考点
- `trap`：易错点
- `analysis`：具体分析
- `memoryTip`：复盘提示
- `reviewStatus`：复盘状态
- `tags`：方便搜索的标签
